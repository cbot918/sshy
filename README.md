# SSHY
a convenient ssh client for practice purpose
# .env
for example
- URL="127.0.0.1:22"
- USER="userr"
- PASSWORD="abcde12345"

# Run
```
go run .
```
or 
```
go build . && ./sshy
```
# SUPPORT COMMAND
```
ls
mkdir
touch 
```
etc
# UNSUPPORT COMMAND
```
cd
clear
```
